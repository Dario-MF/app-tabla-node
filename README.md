# APP en Node generador de tablas de Multiplicar.

```
Opciones:
      --help     Muestra ayuda                                                   [booleano]
      --version  Muestra número de versión.                                      [booleano]
  -b, --base     Base de tabla multiplicar.                            [número] [requerido]
  -l, --listar   Selecciona si quieres ver la tabla en consola. [booleano] [defecto: false]
  -h, --hasta    Número maximo de multiplicaciones.                  [número] [defecto: 10]

```
